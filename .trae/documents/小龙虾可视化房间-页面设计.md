# OpenClaw 像素办公室｜页面设计说明（Desktop-first）

## 全局样式（Global Styles）
- 主题：深色像素风（便于突出像素动画与状态色）
- 背景色：#0B1020；卡片：#121A33；边框：#243055
- 强调色：专注 #4F8CFF｜会议 #B46CFF｜休息 #35D07F｜下班 #8A93A6｜告警/阻塞 #FF5A6A
- 字体：中文无衬线 + 像素标题字体（仅标题/徽标使用）；正文 14–16px，标题 18/22/28
- 按钮：主按钮实心、次按钮描边；hover 提亮 6% + 轻微上浮（translateY(-1px)）
- 动效：面板切换 160ms ease-out；弹层 200ms；避免干扰场景动画

## 页面 1：看板首页（/）

### Meta Information
- Title：OpenClaw 像素办公室｜今日状态看板
- Description：龙虾按状态在像素办公室中移动，记录每日工作小记。
- OG：og:title/og:description + 预览图（像素办公室截图）

### Layout
- 总体：CSS Grid 三栏（左：状态面板；中：场景；右：小记与列表），桌面优先。
- Grid：`240px 1fr 360px`；最小宽度 < 1100px 时右栏下移为第二行。
- 间距：8/12/16 作为基础节奏；卡片统一 12px 内边距。

### Page Structure
1. 顶部栏（Header）
2. 三栏主体（Main Grid）

### Sections & Components
1) 顶部栏
- 左：产品名 OpenClaw + 日期（YYYY-MM-DD）
- 右：进入“小记历史”按钮

2) 左栏：状态面板（Status Panel）
- 当前状态卡：状态徽章、开始时间、今日切换次数
- 状态按钮组：专注/会议/休息/下班（单选）
- 交互：点击即切换；若今日小记未保存，仍允许切换但保留自动保存

3) 中栏：办公室像素场景（Scene Canvas）
- Canvas/舞台区域：固定 16:9，居中适配
- 热点区域标注（弱提示）：工位/会议室/休息区/门口（仅 hover 显示名称）
- 龙虾精灵：
  - 行走：从当前区域插值移动到目标区域
  - 到达：播放对应待机动画（例如敲键盘/开会点头/打盹/离开画面边缘）
- 状态与场景映射：状态决定目标区域、移动速度、停留动画

4) 右栏：今日工作小记（Today Note）
- 输入区：标题（可选）+ 正文 textarea（必填）
- 自动保存提示：显示“已保存/保存中/保存失败（本地）”与最后保存时间

5) 右栏：最近小记列表（Recent Notes）
- 列表项：日期 + 状态色点 + 50 字摘要
- 点击：跳转到 /notes 并定位对应日期

## 页面 2：小记历史页（/notes）

### Meta Information
- Title：OpenClaw 小记历史｜回顾与编辑
- Description：按日期回顾、筛选与编辑每日工作小记。

### Layout
- 左右布局：左侧筛选/日期选择，右侧详情与编辑。
- Grid：`320px 1fr`；窄屏时改为上下布局。

### Page Structure
1. 顶部栏（返回看板）
2. 左侧：日期与筛选
3. 右侧：小记详情

### Sections & Components
1) 顶部栏
- 返回“看板首页”按钮 + 页面标题

2) 左侧：日期浏览与筛选
- 日期列表：按周/月分组的简化列表（默认展示最近 30 天）
- 状态筛选：全部/专注/会议/休息/下班
- 选择某日：右侧加载该日小记

3) 右侧：小记详情维护
- 详情头：日期、状态徽章、更新时间
- 编辑区：标题与正文可编辑，保存按钮（手动保存）
- 删除按钮：二次确认后删除（仅本地数据）

---

## 响应式与可用性补充
- 键盘：状态按钮可 Tab 聚焦；Ctrl/Cmd+S 在编辑区触发保存
- 空状态：无小记时提示“今天写点什么吧”；无历史时提示“还没有记录”
- 性能：场景动画与表单渲染解耦；面板更新不阻塞动画帧
